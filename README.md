# Repeat CA Starter

This is a starter app for the repeat assignment for Web App Development 2.

To use this starter:

- Download the code from this repository

- Run the Movies API:
    - In your terminal, navigate to the *movies-api* folder
    - Run the command `npm install`
    - To start the server, run `npm run dev`
    
- Run the React Movies app:
    - In your terminal, navigate to the *react-movies* folder
    - Run the command `npm install`
    - To start the React app, run `npm run dev`
    
Create your own GitHub repository for the repeat assignment and push the changes you make to that repository.
